Cohevium
=======

A simple but extensible content manager. 


Motivation
==========


Install
=======
To install use the npm:

	npm install altenia/cohevium


Configure
=========
The server configuration file is located in `conf/coheviu.conf.json`
The content configuration file is located in `cohevium-content/cohevium-items.json`


Modules
=======
The modules are located in `cohevium_modules` folder


Start
=====

	node coheviumsvr.js

To run as backround servier in *nix platform

	nohup node coheviumsvr.js


Open browser to the address:
http://localhost:8080/cohevium/content/about.html
